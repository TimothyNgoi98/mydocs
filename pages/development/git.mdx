# Git

## Frequently used


### Repo
```bash
# intialise a repo 
git init 
```

### Branch

```bash
# move to a branch that already exists 
git checkout <branch_name>
# create and move to a new branch
git checkout -b <branch_name>
# delete a branch
git branch -d <branch_name>
# rename a branch
git branch -m <renamed_branch>
```

### Add

```bash
# add all changed files
git add .
# add a specific file 
git add <file_relative_path>
```

### Commit

```bash
# commit changes
git commit -m "commit_message"
```

### Push
```bash
# push a branch which is already connected and exists in remote
git push
```
